# Triage 标签

技能内部使用五个标准 triage 角色。本文件将这些角色映射到本仓库 Issue 系统中实际使用的标签字符串。

| 技能中的标签           | 本仓库使用的标签  | 含义                       |
| ---------------------- | ----------------- | -------------------------- |
| `needs-triage`         | `needs-triage`    | 维护者需要评估此 Issue      |
| `needs-info`           | `needs-info`      | 等待报告者补充更多信息      |
| `ready-for-agent`      | `ready-for-agent` | 已充分描述，可供 AFK agent 领取 |
| `ready-for-human`      | `ready-for-human` | 需要人工实现                |
| `wontfix`              | `wontfix`         | 不予处理                    |

当技能提到某个角色时（如"打上 AFK-ready triage 标签"），请使用本表右侧对应的标签字符串。

如需自定义，直接编辑右侧列即可。
